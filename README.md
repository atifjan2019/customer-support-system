# ISP Support System

## Getting Started

1. **Start MySQL**: Make sure XAMPP MySQL is running.
2. **Start Servers**: Double-click `start-dev.bat`.

## Access

- **Frontend**: [http://localhost:5173](http://localhost:5173)
- **Backend API**: [http://localhost:8000](http://localhost:8000)

## Default Credentials

**Super Admin**
- Email: `admin@isp.com`
- Password: `password`

## Project Structure

- `backend/`: Laravel 12 API
- `frontend/`: React + Vite

## Mobile App (NativePHP)

NativePHP Mobile scaffolding is available in `backend/nativephp/`.
See `docs/NATIVEPHP_MOBILE.md` for setup, prerequisites, and run instructions.
